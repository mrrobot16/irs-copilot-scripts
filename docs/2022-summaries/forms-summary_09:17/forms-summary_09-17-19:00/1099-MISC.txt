This document provides instructions and information regarding Form 1099-MISC, a tax form used to report miscellaneous income. It explains that Copy A of the form, which is provided for informational purposes, should not be printed and filed because it is not scannable by the IRS. However, Copy B and other copies of the form can be downloaded and printed for distribution to recipients. The document also provides information on how to order official IRS information returns and how to file electronically using the IRS Filing Information Returns Electronically (FIRE) system or the IRS Affordable Care Act Information Returns (AIR) program. It includes instructions for recipients on how to report the income and provides contact information for assistance with reporting.
