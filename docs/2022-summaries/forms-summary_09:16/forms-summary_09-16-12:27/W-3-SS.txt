This document provides instructions for employers on how to file Forms W-2 and W-3 electronically with the Social Security Administration (SSA) or by mail. It states that employers can create fill-in versions of Forms W-2 and W-3 on the SSA's website, which can be used for filing with the SSA, distributing to employees, and for record-keeping purposes. However, it notes that the online version of the forms printed from the website is not scannable and should not be filed with the SSA. It also provides information on how to order official IRS information returns, such as Forms W-2 and W-3, for filing with the SSA. The document includes instructions for completing Form W-3SS, which is used as a transmittal for paper copies of Forms W-2AS, W-2CM, W-2GU, or W-2VI. It explains the purpose of Form W-3SS and provides instructions on when and where to file it. The document also includes information on e-filing options and provides contact information for filing Copy 1 of Form W-3SS with state, city, or local tax departments in specific locations.
